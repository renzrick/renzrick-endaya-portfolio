Challenges

  • I want to create a social platform that allows for users to create various reviews on an album, having the user log which albums they want to listen to later, as well as have the ability to share the albums they have already listened to

  • I also want to enable my user testers to successfully go through the curated user flow, which depicts the process of signing up for an account (or logging in to a pre-existing account) — then going on to create a review for a specific album.

UX Design

  • To start off, a flow chart was created to display the process of a user creating an account, then being able to log a music review within the account created.

  • A User persona and scenario were created to give viewers of the project an insight of a how a typical user would come across and act upon the overall website

  • The wireframes and prototypes produced were all created within Adobe XD, and were all designed with the intent of having the user experience be as straightforward and aesthetically pleasing as possible.

Testing

After taking the reviews into consideration from the users in my testing group, I felt that it was a common occurrence that they all suggested that I make changes specifically within the design aspect of my prototype. Firstly, I chose to add hover overlays to the pictures of the album covers — further distinguishing them which would make it easier for the user to identify the album being shown, especially for those that have never heard of a certain artist. This feature was applied to the rest of the albums within the column.
